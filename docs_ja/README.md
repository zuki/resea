# Resea

Resea *[ríːseə]* は0から書いたマイクロカーネルベースのオペレーティングシステム
です。設計全体を直感的に理解でき、システムのカスタマイズが容易であり、機能拡張が
楽しくなるよう、魅力的な開発者体験と*ハッカブル*であることを目指しています。

## 特徴

- *x86_64 (SMP)をサポートする純粋な*マイクロカーネル
- TCP/IPサーバ
- デバイスドライバ（PS/2キーボード、e1000ネットワークカード、テキストモード画面）
