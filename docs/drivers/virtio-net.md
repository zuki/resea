# virtio-net
A network device driver for virtio-net version 1.x (so-called *modern device*) and 0.x (*legacy device*).

# How to Activate on QEMU
```
$ make run VIRTIO_NET=1
$ make run VIRTIO_NET=1 VIRTIO_MODERN=1
```
